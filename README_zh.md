# ZAI-Skills

[English](./README.md) | [简体中文](./README_zh.md) | [日本語](./README_ja.md)

一套专门为 **Z.AI (Z Code)** 平台及其核心 MCP 生态系统（Vision, Search 和 Zread）设计的高性能 [Agent Skills](https://github.com/skillcreatorai/Ai-Agent-Skills) 集合。

## 🌟 为什么选择 ZAI-Skills？

这些技能经过优化，增强了 Z.AI 智能套件的三大支柱，为高保真技术输出提供高级指令。

## 🚀 可用技能

- **`zai-orchestrator` (新！)**：“大脑”技能。协调 Vision, Search 和 Zread 来解决复杂的、多步骤的工程任务。
- **`vision-expert`**：针对 Z.AI Vision 的高级 UI 转代码、技术图表分析和视觉错误诊断。
- **`search-expert`**：使用 Z.AI Web Search Prime 进行高精度技术研究和结构化信息综合。
- **`zread-expert`**：通过 Zread 进行资深架构师级别的代码库分析、结构审计和语义搜索。

## 🛠 使用方法

直接将技能安装到您的 Z Code, Claude Code 或 Cursor 环境中：

```bash
npx skills add https://github.com/tianxiao1430-jpg/zai-skills --skill zai-orchestrator
```
