# Z.AI Expert Skills Library

A collection of specialized [Agent Skills](https://github.com/skillcreatorai/Ai-Agent-Skills) designed specifically for the **Z.AI (Z Code)** ecosystem. These skills enhance the performance of the three core Z.AI MCP tools: Vision, Search, and Zread.

## Skills Included

- **vision-expert**: Advanced UI-to-Code and technical diagram analysis.
- **search-expert**: High-precision web research and source synthesis.
- **zread-expert**: Deep codebase analysis and PR/Issue investigation.

## Usage

You can add these skills to your Z Code environment by cloning this repo into your skills directory or using a skill manager.

```bash
# Example for local Z Code setup
npx skills add https://github.com/tianxiao1430-jpg/zai-expert-skills --skill vision-expert
```

## Configuration

Ensure your `mcp.json` is configured with the official Z.AI MCP endpoints.
